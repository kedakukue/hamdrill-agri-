# Hamdrill Enterprises — Website

Flask site for Hamdrill Agri (poultry, horticulture, sustainability, contact).

## Local development

```bash
python -m venv venv
venv\Scripts\activate          # Windows
pip install -r requirements.txt
copy .env.example .env         # if you don't have .env yet
python app.py
```

Open http://127.0.0.1:5001

## Contact form (Formspree)

1. Create a free account at [formspree.io](https://formspree.io).
2. **New form** → set the notification email (e.g. `administration@hondehgroup.com` or your Hamdrill address).
3. Open **Integration** and copy the form URL, e.g. `https://formspree.io/f/abcxyzxy`.
4. Paste it in `.env`:

   ```env
   FORMSPREE_ACTION=https://formspree.io/f/abcxyzxy
   ```

5. Restart `python app.py` and test **Contact** → **Send Message**.

`SECRET_KEY` in `.env` is already set for local use. **Never commit `.env`** (it is in `.gitignore`).

## Deploy free with GitHub + Render

GitHub **Pages** only hosts static files; this app needs Python, so we host on [Render](https://render.com) (free tier) and keep the code on **GitHub**.

### 1. Push code to GitHub

1. Install [Git](https://git-scm.com/) if needed.
2. Create a new repository on GitHub (e.g. `hamdrill-agri`).
3. In this folder:

   ```bash
   git init
   git add .
   git commit -m "Initial Hamdrill site"
   git branch -M main
   git remote add origin https://github.com/YOUR_USER/hamdrill-agri.git
   git push -u origin main
   ```

### 2. Deploy on Render (free)

1. Sign up at [render.com](https://render.com) (use **Sign in with GitHub**).
2. **New +** → **Blueprint** (or **Web Service** → connect the repo).
3. Select your `hamdrill-agri` repository. Render detects `render.yaml`.
4. In the service **Environment** tab, set:

   | Key | Value |
   |-----|--------|
   | `FORMSPREE_ACTION` | `https://formspree.io/f/YOUR_REAL_ID` |
   | `CONTACT_EMAIL` | (optional) display email on the site |
   | `SECRET_KEY` | auto-generated by Render if using blueprint |

5. Deploy. Your site will be at `https://hamdrill-agri.onrender.com` (name may vary).

Free Render services sleep after inactivity; the first visit may take ~30 seconds to wake up.

### 3. Custom domain (optional)

In Render → your service → **Settings** → **Custom Domain**, follow the DNS instructions.

## Environment variables

| Variable | Purpose |
|----------|---------|
| `SECRET_KEY` | Flask session signing (required in production) |
| `FORMSPREE_ACTION` | Formspree form POST URL |
| `CONTACT_EMAIL` | Shown on Contact / footer |
| `WHATSAPP_NUMBER` | Digits only, e.g. `263242705863` |
| `WHATSAPP_DISPLAY` | Formatted phone for display |
| `FLASK_DEBUG` | `1` local only; `0` on Render |
| `PORT` | Set by Render automatically |
